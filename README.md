## 🚀 Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/ApiCartOps/apicartops.github.io.git
cd apicartops.github.io
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

3. Navigate to `http://localhost:8000` and view the site.

## 📝 Customization

### Adding New Projects

Edit `index.html` and add a new project card in the projects grid:

```html
<div class="project-card">
    <div class="project-header">
        <h3>Project Name</h3>
        <a href="https://github.com/ApiCartOps/project-name" target="_blank" class="github-link">View on GitHub</a>
    </div>
    <p class="description">Project description here.</p>
    <div class="tech-stack">
        <span class="tech">Tech1</span>
        <span class="tech">Tech2</span>
    </div>
</div>
```

### Styling Customization

Edit `styles.css` to customize:
- Color scheme (CSS variables in `:root`)
- Typography
- Spacing and layout
- Responsive breakpoints
